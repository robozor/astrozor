# Vítej v Astrozoru

**Astrozor** je platforma pro českou astronomickou komunitu — amatéry, profesionály, vzdělavatele a citizen-science nadšence. Sdružuje na jednom místě:

- **Mapa hvězdáren** se vrstvou světelného znečištění a kvalitou oblohy
- **Akce** (pozorování, srazy, konference) s registracemi a kalendářem
- **Citizen Science kampaně** (Zooniverse integrace pro klasifikaci dat)
- **Vědecké články** s DOI (přes Zenodo), MathJax rovnicemi, interaktivními grafy
- **Projekty** s GitHub integrací (issues, řešitelé, leaderboardy)
- **Diskuze a komentáře** pod každým článkem / projektem / akcí
- **Notifikace** (web push, Discord webhooks, Mastodon cross-posting)

## Pro koho

- **Amatérští astronomové** — najdi nejbližší tmavou oblohu, přihlas se na pozorování, sdílej zápisky z noci
- **Profesionálové** — publikuj recenzované články s DOI, integruj s vědeckými infrastrukturami (Zooniverse, Zenodo)
- **Pedagogové** — připrav citizen-science kampaň pro studenty, naviguj k veřejně přístupným datům
- **Vývojáři** — koukni do projektů, vezmi si issue, přispěj na GitHubu — Astrozor ti to ukáže na leaderboardu

## Klíčová filozofie

- **Otevřená data, otevřená vědecká kultura.** Default licence CC BY 4.0, DOI přes Zenodo, RSS/Atom feedy, sitemap pro indexery.
- **Žádné lock-in.** Texty exportuješ jako markdown, články jako HTML bundle, data přes REST API.
- **Self-hostable.** Celý stack v Docker compose. Komunitní instance + privátní instance pro tvou organizaci jsou ekvivalentní.
