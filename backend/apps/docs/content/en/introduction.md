# Welcome to Astrozor

**Astrozor** is a platform for the Czech astronomy community — amateurs, professionals, educators, citizen-science enthusiasts. In one place:

- **Observatory map** with light-pollution and sky-quality overlays
- **Events** (observations, meetups, conferences) with registration and a calendar
- **Citizen Science campaigns** (Zooniverse integration for data classification)
- **Scientific articles** with DOI (via Zenodo), MathJax equations, interactive charts
- **Projects** with GitHub integration (issues, assignees, leaderboards)
- **Discussion and comments** under every article / project / event
- **Notifications** (web push, Discord webhooks, Mastodon cross-posting)

## Who it's for

- **Amateur astronomers** — find the nearest dark sky, sign up for an observation, share your night log
- **Professionals** — publish peer-reviewed articles with DOI, integrate with research infrastructure (Zooniverse, Zenodo)
- **Educators** — set up a citizen-science campaign for students, point them at public datasets
- **Developers** — browse projects, claim an issue, contribute on GitHub — Astrozor tracks you on the leaderboard

## Core philosophy

- **Open data, open scientific culture.** Default license CC BY 4.0, DOI via Zenodo, RSS/Atom feeds, sitemap for indexers.
- **No lock-in.** Texts export as markdown, articles as HTML bundles, data via REST API.
- **Self-hostable.** The whole stack runs in Docker compose. Community instance and your own private instance are equivalent.
